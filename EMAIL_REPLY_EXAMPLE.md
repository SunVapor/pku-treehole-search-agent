## 回复内容解析

### 1. 搜索过程部分
```markdown
**搜索过程** (3 次搜索):

1. `AI 机器学习 课程` - 搜索AI和机器学习相关课程推荐
2. `人工智能 课程 推荐` - 搜索更多人工智能相关课程推荐
3. `AI引论 机器学习 课程测评` - 搜索AI引论和机器学习具体课程测评
```

**说明**：
- 显示 LLM 进行了几次搜索（1-3次）
- 每次搜索使用的关键词
- LLM 选择这些关键词的原因

### 2. 回答内容
- 综合所有搜索结果生成的完整回答
- 使用 Markdown 格式，清晰易读
- 包含具体的课程信息、建议等

### 3. 参考来源
```markdown
**参考来源**: 34 个帖子
```

显示总共找到多少个相关帖子作为信息来源。

## 为什么显示搜索过程？

### 透明度 🔍
让你清楚知道 AI 是如何工作的，进行了哪些搜索。

### 可信度 ✅
通过展示搜索关键词，你可以判断信息来源是否可靠。

### 迭代优化 🔄
如果回答不满意，你可以看到搜索了哪些关键词，下次可以直接用手动模式指定更精确的关键词。

## 邮件格式优势

- ✅ **Markdown 格式**：在邮件客户端中自动渲染为美观的 HTML
- ✅ **结构清晰**：标题、列表、代码块等格式完整
- ✅ **信息全面**：包含搜索过程、回答、来源等完整信息
- ✅ **便于保存**：可以直接转发、保存到笔记软件
